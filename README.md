## 📖 Project Overview

This project performs **Exploratory Data Analysis (EDA)** and **Feature Engineering** on a real-world salary dataset. The goal is to discover meaningful patterns, identify key factors influencing salaries, handle data quality issues, and engineer features that enhance the dataset for downstream machine learning tasks.

Whether you're a data science enthusiast or a professional, this project offers practical insights into salary trends, distributions, and correlations across various dimensions.

## 📊 Dataset Information

- **Dataset Name:** `Salary Dataset.csv`
- **Key Features Include:**
  - Job Title / Role
  - Experience Level
  - Employment Type
  - Company Size & Location
  - Salary (in USD)
  - Number of Salaries Reported

## 📈 EDA Highlights

- ✅ Analyzed the **distribution of salaries** across job roles and experience levels
- ✅ Explored the **relationship between salary and number of salaries reported**
- ✅ Identified and handled **outliers** in salary data
- ✅ Investigated **company size and location** impact on compensation
- ✅ Explored **correlations** between numerical features using heatmaps

---

## 🛠️ Feature Engineering Steps

- 🔄 Converted **categorical variables** (job title, company size, employment type) into numerical formats
- 🕒 Extracted and engineered **meaningful features** from existing columns
- 🧹 Handled **missing values** using appropriate imputation strategies
- 📏 Applied **scaling and normalization** to numerical columns
- 🔍 Created **interaction features** to capture combined effects

## 🚀 How to Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AbhishekDhawan07/EDA_and_Feature_Engineering_Project_on_Salary_Dataset.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd "EDA and Feature Engineering Project on Salary Dataset"
   ```

3. **Install required dependencies:**
   ```bash
   pip install pandas numpy matplotlib seaborn jupyter
   ```

4. **Launch the Jupyter Notebook:**
   ```bash
   jupyter notebook "EDA and Feature Engineering on Salary Dataset.ipynb"
   ```

5. **Run all cells** to reproduce the full analysis and visualizations.
